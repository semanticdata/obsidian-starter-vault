# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.4.0] - 2025-08-17

- Replaced unmaintained Home tab plugin with Homepage plugin
- Added Welcome page for user onboarding
- Enhanced journaling system with improved templates
- Added nldates-obsidian plugin for natural language dates
- Updated CSS snippets for tables and internal links
- Optimized hotkeys and workspace layout
- Expanded README and documentation

## [1.3.0] - 2024-02-23

- Reorganized CSS snippets with proper categorization
- Improved typography, tables, and frontmatter styling
- Added Calendar plugin styling and task wrapper improvements
- Added dark PDF export CSS and compact UI elements
- Added Tag Wrangler plugin and updated community extensions
- Expanded daily notes setup documentation

## [1.2.0] - 2023-11-15

- Added Table Editor Obsidian plugin
- Enhanced CSS snippets and styling
- Updated community extensions
- Expanded daily notes documentation and customization options

## [1.1.0] - 2023-10-27

- Added Markdown Demo page
- Integrated new community plugins
- Refined CSS snippets for better appearance

## [1.0.0] - 2023-08-28

- Initial release based on Obsidian v1.4.5
- Implemented Minimal theme
- Preinstalled essential community plugins
- Included custom CSS snippets
- Optimized workspace layout and documentation
