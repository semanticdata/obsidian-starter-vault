# Plugins Installed

This vault includes several plugins pre-configured. They're already installed - you don't need to do anything.

## Core Plugins (Built into Obsidian)

| Plugin | What It Does |
|--------|--------------|
| **Daily notes** | Creates a note for each day automatically |
| **Templates** | Lets you use templates from the templates folder |
| **Graph view** | Shows how your notes connect visually |
| **Backlinks** | Shows which notes link to the current note |
| **Outgoing links** | Shows which notes the current note links to |
| **Tags** | View and navigate all tags in your vault |

## Community Plugins (Installed)

| Plugin | What It Does | Why It's Here |
|--------|--------------|---------------|
| **Homepage** | Opens a specific note when you launch Obsidian | Opens Welcome.md automatically |
| **Omnisearch** | Searches across all notes, including typos | Find notes even when you misspell |
| **Linter** | Auto-formats your notes when you save | Keeps formatting clean (`Ctrl + S`) |
| **Advanced Tables** | Makes creating and editing tables easier | Tables don't fight back |
| **Natural Language Dates** | Converts `@today` to today's date | Type `@today` instead of the date |
| **Tag Wrangler** | Rename and manage tags | Keep tags organized |
| **Editor Shortcuts** | Adds keyboard shortcuts from code editors | Navigate like VS Code |
| **Doubleshift** | Quick command launcher | Press `Shift` twice to search |

## Managing Plugins Later

If you want to add, remove, or configure plugins:

1. Open Settings (`Ctrl` + `,`)
2. Click **Community plugins** on the left
3. Browse available plugins or manage installed ones

**Note**: You can safely disable any plugin here if you don't use it. Your notes won't be affected.

---

**Notes**:
- Plugins run in the background - you won't notice most of them
- Check plugin settings if you want to customize things
- The Obsidian forum has good plugin discussions if you need help
